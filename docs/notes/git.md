# Git 笔记

## 常用命令

```bash
git init          # 初始化仓库
git add .         # 暂存所有改动
git commit -m ""  # 提交
git push          # 推送到远程
git pull          # 拉取远程更新
git status        # 查看状态
git log --oneline # 查看提交历史
```

## 分支操作

```bash
git branch feature     # 创建分支
git checkout feature   # 切换分支
git merge feature      # 合并分支
```
